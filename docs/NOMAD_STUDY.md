# 🕯️ 유목민 공부법 가이드

> **"왜 이 종목이 올랐을까?"를 매일 공부하여 시장의 흐름을 읽으세요.**

---

## 📚 개요

유목민 공부법은 매일 장 마감 후, 시장을 주도한 상한가 및 거래량 급증 종목을 분석하는 훈련법입니다. Hantubot은 이 과정을 자동화하여 시간을 절약해줍니다.

### 주요 기능
- **자동 수집**: 상한가 및 거래량 1,000만주 이상 종목 자동 필터링
- **뉴스 매칭**: Naver 뉴스에서 관련 기사 자동 검색
- **AI 요약**: Google Gemini AI가 상승 이유를 3줄로 요약
- **데이터베이스**: 모든 기록은 `data/study.db`에 영구 저장

---

## 🚀 사용 방법

### 1. 자동 실행 (기본)
Hantubot 엔진이 켜져 있다면, **매일 오후 3시 35분**경 자동으로 실행됩니다.
- Discord로 "오늘의 공부 종목" 알림이 전송됩니다.
- 데이터베이스에 자동으로 저장됩니다.

### 2. 수동 실행 (테스트)
장 마감 후 수동으로 실행하고 싶다면 다음 명령어를 입력하세요:

```bash
python -m hantubot.reporting.study --force
```

### 3. 대시보드 조회
수집된 데이터를 시각적으로 확인하고 공부할 수 있습니다.

```bash
streamlit run dashboard/app.py
```
- 브라우저에서 `http://localhost:8501` 접속
- 날짜별 조회, 종목 필터링, AI 요약 확인 가능

---

## ⚙️ 설정 방법

`.env` 파일에서 다음 항목을 설정해야 AI 요약이 작동합니다.

```env
# Gemini API 키 (필수)
GEMINI_API_KEY="AIzaSy..."

# 모드 설정
STUDY_MODE=sqlite
LLM_BATCH_SIZE=10
```
